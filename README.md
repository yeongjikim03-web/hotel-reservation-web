# 호텔 예약 시스템 (DB, 웹 프로젝트)

##  실행 방법

1. MySQL 실행 후 `hotel.sql` 파일을 실행하여 데이터베이스를 생성한다.
2. Tomcat 서버를 실행한다.
3. 브라우저에서 아래 주소로 접속한다.

http://localhost:8080/프로젝트명/mainhome.jsp

## 주요 기능

- 사용자는 객실 소개를 통해 모든 객실을 확인할 수 있다.
- 사용자는 객실 검색을 통해 날짜, 인원, 객실 타입을 선택하여 조건에 맞는 객실을 찾을 수 있다.
- 예약 시 `reservation` 테이블에 예약 정보가 저장되며,
  해당 객실의 `is_available` 값이 0으로 변경되어 예약 불가 상태가 된다.
- 사용자는 마이페이지를 통해 예약 내역을 조회할 수 있다.
- 사용자는 마이페이지의 예약 내역을 통해 예약을 취소할 수 있다.
- 예약 취소 시 `reservation` 테이블의 `status` 값이 `canceled`로 변경되며,
  동시에 `room` 테이블의 `is_available` 값이 1로 변경되어 다시 예약 가능 상태로 복구된다.
- 마이페이지에서 사용자의 멤버십 정보와 고객 정보를 볼 수 있다.
